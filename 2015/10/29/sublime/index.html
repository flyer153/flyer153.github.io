sublime text3 使用技巧

我用的是mac下的sublime text3，和win大同小异。记录一下我平常用的插件和快捷键以及一些技巧。

安装插件

直接下载安装包解压缩到Packages目录（sublime text3->preferences->Browse Packages）

删除插件

command+shift+P,输入remove package
输入要删除的package
回车

快捷键

command+L 选择整行（按住-继续选择下行）
command+Shift+K(shhift+del) 删除整行，
command + KK 从光标处删之行尾，command+K Backspace 从光标处删除至行首
command+Shift+D 复制光标所在整行，插入在该行之前
command+D 选词 （按住-继续选择下个相同的字符串，再按，可跳到相应的方法定义处
command+Shift+M 选择括号内的内容（按住-继续选择父括号）
command+/ 注释整行（如已选择内容，同“command+Shift+/”效果）
command + alt + / 取消注释
command+Shift+UP 与上行互换 command + shift + up: 列模式编辑
command + R 跳转当前页的目标方法
command+K + U 大写
command+K + L 小写
鼠标中间 列模式编辑
command+Shift+[] 代码折叠
command+k command+1: 折叠所有代码
command + K,B 打开侧边栏
command + 回车：　　 光标后插入行，　command+Shift+Enter 光标前插入行
command + m: 匹配括号
command +z, y: 撤销，恢复撤销
alt + .: 闭合当前标签
command+F2: 设置书签
F2: 下一个书签
Shift+F2: 上一个书签
command + p: 即时的文件切换
command + shift + a: 选择标签内的内容
command + 单击： 多行随意位置添加光标
alt + F3( mac: command + command + g): 选择页面中所有相同的词
command + F3: 跳转到下一个选中的词
command+Shift+P Set Syntax:html : 设置文件类型
Shift + 右键: 连续多行光标选中 (by Gary Gauh)

窗口操作：

command+w p: 在两个分割窗口之间来回切换。
command+w j: 跳到下面的分割窗
command+w h: 跳到左边的分割窗。
command+w k: 跳到上面的分割窗。
command+w l: 跳到右边的分割窗。

折叠：

zo 将光标所在处的折叠打开。open。
zc 将光标所在处已打开的内容再度折叠起来。close。
zr 将全文的所有折叠依层次通通打开。reduce。
zm 将全文已打开的折叠依层次通通再折叠起來。more。
zR 作用和 zr 同，但会打开含层次折叠（折叠中又还有折叠）的所有折叠。
zM 作用和 zm 同，但对于层次折叠也有作用。
zi 这是个切换，是折叠与不折叠指令间的切換。
zn 打开全文的所有折叠。fold none。
zN 這是 zn 的相对指令，回复所有的折叠。
